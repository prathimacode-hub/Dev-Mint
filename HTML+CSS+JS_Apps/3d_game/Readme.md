Instructions on how to start the game:

The game is built on engines using three.js and Box2dWeb.min.js, so to start the server follow the instructions:


1-Navigate to 3d_game directory
2-Start 'python -m SimpleHTTPServer' in your command-line (for python 3.0 and above type 'python -m http.server' in your shell)
3-Open 'localhost:8000' in your browser


Intructions on how to play the game have been incorporated in the game itself and can be accessed using long press 'I' command


Enjoy!
